154 задача. Дан массив с числами. Создайте из него новый массив, где останутся лежать только положительные числа. Создайте для этого вспомогательную функцию isPositive(), которая параметром будет принимать число и возвращать true, если число положительное, и false - если отрицательное. 

        function isPositive(num) {
            if (num >=0) {
                return true;
            } else {
                return false;
            }
        }
