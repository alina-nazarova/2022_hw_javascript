82 задача. Дан массив с элементами 4, 2, 5, 19, 13, 0, 10. Найдите квадратный корень из суммы кубов его элементов. Для решения воспользуйтесь циклом for. 

        let arr = [4, 2, 5, 19, 13, 0, 10];
        let sum = 0;
        for (i=0; i < arr.length; i++){
            sum += Math.pow(arr[i], 3)
        }
        document.write(Math.sqrt(sum));
